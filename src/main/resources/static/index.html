# Welcome

Hi, thanks for stopping by.

I use this to study some technologies I might be interested in or to test things in combination.

I might as well decide to do some blog posts here inspired by Scott Hanselman.

This is developed with VuePress.

It runs Tomcat as a web server using spring boot. (Which is not really used in the project so far).

Then everything goes into docker container and I deploy at the cloud.

Source code is here.

I use Let's Encrypt for the SSL certificates and nginx for reverse proxy, gzip, ssl, h2 and a bit more.

Every new commit to master triggers a build via CircleCi and deploys a new image to the docker hub.
